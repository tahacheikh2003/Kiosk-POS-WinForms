Kiosk POS Management System

A desktop Point of Sale (POS) and Inventory Management System developed using C# WinForms and SQL Server.

This project simulates a small kiosk/shop management system with user authentication, product management, stock tracking, sales processing, and shift management.

How to Run the Project
1. Setup the Database

Open SQL Server Management Studio and run:

Data Base/KioskDB2_full.sql

This will automatically create:

Database KioskDB2

Tables

Views

Functions

Stored Procedures

2. Configure SQL Server

The application expects the SQL instance:

.\SQLEXPRESS

If your SQL Server instance is different, update the connection in the DataLayer class.

3. Run the Application

Open the project:

Projet_Data_Base_Windows Form

Run the WinForms application.
